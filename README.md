# Battleship

Everyone knows this classic game. You place your ships, and try to sink the enemy's ships. Good luck!

## Requirements

npm v8.11.0 or up
TypeScript ^4.7.4

## Installation

```bash
git clone https://github.com/adi-lux/Battleship.git foldername
cd foldername
npm install
npm run start
```
